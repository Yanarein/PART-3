# PART-3 SHOPPING CART

Shopping Cart System 
Project Overview
This is a step-by-step Object-Oriented Programming (OOP) project built in C#.
It simulates a shopping cart system with product management, cart operations, and an interactive console menu.
The project is developed progressively from Day 1 to Day 5 to demonstrate structured learning and software development practice.
--------------------------------------------------------------------------------
Features
Day 1 – Product System
• Product class creation
• Stores:
    ◦ ID
    ◦ Name
    ◦ Price
--------------------------------------------------------------------------------
Day 2 – Shopping Cart System
• ShoppingCart class
• List-based product storage
• Add products to cart
• Display receipt-style output
• Calculate total price
--------------------------------------------------------------------------------
Day 3 – Quantity System
• Added CartItem class
• Quantity per product
• Subtotal calculation
• Remove product feature
--------------------------------------------------------------------------------
Day 4 – Checkout System
• Payment handling
• Discount calculation
• Change computation
• Final receipt output
--------------------------------------------------------------------------------
Day 5 – Interactive System
• Menu-driven interface
• View products
• Add product using ID
• Input quantity
• View cart
• Checkout system
• Exit option
